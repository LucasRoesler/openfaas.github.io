A Serverless Framework for Docker and Kubernetes

view on github YouTube Demo

With OpenFaaS you can package anything as a serverless function - from Node.js to Golang to CSharp, even binaries like ffmpeg to ImageMagick.

You can try OpenFaaS in 60 seconds or write and deloy your first Python function in around 10-15 minutes.

So bring your laptop, your own on-prem hardware or create a cluster in the cloud. Pick Docker or Kubernetes to do the heavy lifting enabling you to build a scaleable, fault-tolerant event-driven serverless platform for your applications.
